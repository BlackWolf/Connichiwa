Inherits from GCDWebServerResponse : NSObject
Declared in GCDWebServerFileResponse.h
GCDWebServerFileResponse.m

Overview

The GCDWebServerFileResponse subclass of GCDWebServerResponse reads the body of the HTTP response from a file on disk.

It will automatically set the contentType, lastModifiedDate and eTag properties of the GCDWebServerResponse according to the file extension and metadata.

initWithFile:byteRange:

Initializes a response like initWithFile: but restricts the file contents to a specific byte range. This range should be set to (NSUIntegerMax, 0) for the full file, (offset, length) if expressed from the beginning of the file, or (NSUIntegerMax, length) if expressed from the end of the file. The “offset” and “length” values will be automatically adjusted to be compatible with the actual size of the file.

- (instancetype)initWithFile:(NSString *)path byteRange:(NSRange)range

Discussion

This argument would typically be set to the value of the byteRange property of the current GCDWebServerRequest.
