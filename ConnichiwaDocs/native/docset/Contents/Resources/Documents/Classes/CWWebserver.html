Overview

The CWWebserver class represents the local webserver run by Connichiwa in order to run local web applications. It launches the webserver and acts as a bridge between Objective-C and the server, sending and receiving messages from it over the Connichiwa Communication Protocol (Native Layer). Only one instance of a CWWebserver can run in an application - always use the sharedServer method to retrieve that instance.
