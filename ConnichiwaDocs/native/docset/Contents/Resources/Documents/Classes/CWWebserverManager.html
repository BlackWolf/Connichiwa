Inherits from NSObject

Overview

The CWWebserverManager class represents the local webserver run by Connichiwa in order to run local web applications. It is responsible for launching and managing the webserver and acts as a bridge between Objective-C and the server, sending and receiving messages from it over the Connichiwa Communication Protocol (Native Layer). Only one instance of the webserver should be running on a device.
