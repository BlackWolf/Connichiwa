(static) getEventLocation(e, typeopt) → {Point}

Returns the coordinate of a given mouse event or the first touch in a touch event

Parameters:
Name Type Attributes Default Description
e TouchEvent | MouseEvent | jQuery.Event

A valid mouse or touch event or a jQuery event wrapping such an event. Note that events resulting from a "touchend" are not considered valid, as they do not contain any touches
type String <optional>
 "page"

A string representing the type of locarion that should be returned. Can be either "page", "client" or "screen". See Touch Documentation or [MouseEvent Documentation]https://developer.mozilla.org/en-US/docs/Web/API/MouseEvent
Returns:

The coordinate of the event
