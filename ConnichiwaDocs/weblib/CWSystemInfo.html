Namespace: CWSystemInfo

Overview

CWSystemInfo encapsulates some system-related information as far as they are available. It can be used, for example, to access the display's PPI, the browser window resolution or the orientation

This device's display PPI, approximated. JavaScript does not have direct access to the display size and therefore cannot retrieve the PPI value.
